Kontextsensitive Hilfe fuer WordPress-Adminbereich. Verbindet sich beim Aufruf mit dem WordPress-Wiki.

== Description ==

WordPress-Wiki ist hauptsaechlich als kontextsensitive Hilfe gedacht. Mit dem WordPress-Plugin mmyHelp wird ein kleines Fragezeichen in den Kopfbereich des WordPress-Backends eingefuegt. Benoetigt ein User Hilfe mit dem aktuell aufgerufenen Menuepunkt oder einzelnen Funktionen, kann er mit einem Klick auf dieses Fragezeichen das entsprechende Hilfekapitel aus diesem Wiki aufrufen.

== Installation ==

1. Den Ordner `mmyHelp` in den Ordner `/wp-content/plugins/` laden.
2. Aktiviert das Plugin `mmyHelp` in der Plugin-Verwaltung.
